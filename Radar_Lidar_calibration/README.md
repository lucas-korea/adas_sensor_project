## Radar_Lidar_calibration
- point cloud 기반 다중 3d 센서 캘리브레이션 툴
- 수동으로 직접 회전, 이동하여 동시 취득된 두 데이터 간 변환 행렬을 만들어 낼 수 있다.
- 개발환경 : windows10, python3.8
